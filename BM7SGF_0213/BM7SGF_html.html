Git alap parancsok

Git egy elosztott verziókezelő rendszer, melynek alapvető parancsait gyakran használják fejlesztők a projektjeik kezelésére.

Leggyakrabban használt parancsok

git init: Új Git repository inicializálása

git init

git clone: Egy meglévő repository klónozása

git clone repository_url

git add: Fájlok hozzáadása a staging area-hoz

git add file.txt

git commit: Staged fájlok commitolása

git commit -m "Commit message"

git push: Lokális változtatások feltöltése a távoli repository-be

git push origin branch_name

git pull: Frissítés a távoli repository-ból

git pull origin branch_name

git branch: Branch-ek listázása

git branch

git checkout: Branch váltás

git checkout branch_name

git merge: Branch-ek összefésülése

git merge branch_name
