A jQuery népszerű JavaScript könyvtár, mely a HTML kód és a kliensoldali JavaScript közötti kapcsolatot hangsúlyozza.
2006 januárjában jelentette meg a Mozilla Alapítvány népszerű JavaScript evangelistája, John Resig.
A függvénykönyvtár MIT és GNU kettős licenc alatt jelent meg. A jQuery ingyenes, nyílt forrású szoftver.
